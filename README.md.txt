# 🌡️ Smart Monitor ML – Simulation d'anomalies capteurs avec Machine Learning

Ce projet simule un système embarqué intelligent capable de détecter des anomalies de température et d'humidité en utilisant un modèle de Machine Learning. Il inclut une interface graphique conviviale pour l'affichage en temps réel, l'historique des lectures, et un graphique dynamique.

---

## 🧠 Fonctionnalités
- Simulation de capteurs température & humidité
- Prédiction automatique d'anomalie via un modèle Random Forest
- Mode manuel & mode automatique
- Affichage graphique en temps réel (matplotlib)
- Export CSV de l'historique
- Interface graphique développée en `Tkinter`
